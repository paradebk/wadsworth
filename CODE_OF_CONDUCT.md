# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code
of conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Summary

We want every interaction in this project — issues, pull requests,
discussions, code review — to be welcoming, respectful, and focused on
the work. Disagreement on technical matters is expected and healthy;
personal attacks, harassment, and disrespect of any kind are not.

## Scope

This Code of Conduct applies to all project spaces — the GitHub
repository, pull request reviews, issue threads, GitHub Discussions, and
any other forum operated under the project's name. It also applies when
an individual is officially representing the project in public spaces.

## Enforcement

Instances of unacceptable behavior may be reported by opening a GitHub
issue marked confidential, or by contacting the project maintainer
directly at the email listed on the maintainer's GitHub profile. All
complaints will be reviewed and investigated promptly and fairly.

The project maintainer is obligated to respect the privacy and security
of the reporter of any incident.
